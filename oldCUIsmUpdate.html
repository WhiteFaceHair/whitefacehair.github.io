4.0Beta 업데이트 :


- 방화벽 세션 추가

코드스와인과 같은 방식의 솔로세션 제작 기능임.

공개세션에서 활성화 한다면 순식간에 솔로 세션으로 만들어 줌.

방화벽 포트를 막아버리는 거라서 공개 솔로세션과 다르게 비활성화 하기 전 까진 다른 유저가 내 세션에 들어올 수 없음.

새 세션을 찾거나 로딩이 필요한 임무에 들어갈 때 오류가 뜬다면 방화벽 세션을 비활성화 하고 접속해야 함.

ffff와 언노운 에러를 피하고, 사람도 마주칠 순 없을까 해서 만들어 봤음.

근데 이거 쓰는 사람이 있을지는 모르겠다 ㅋㅋ

이 글은 다운로드 링크를 걸어놓는 용도로만 쓰고 내 깃허브 페이지에서 업데이트나 사용법을 볼 수 있게 할 계획임.

원래 4월 달에 글 한 번 엎을려고 했는데 시간이 너무 오래 걸려서 포기함.

6월 달에 종강하고 나서 심심하면 할 듯.

https://download.sysinternals.com/files/PSTools.zip

2.1 업데이트 :


추가 된 것만 보면 별 거 없어 보이는데, 코드 작성할 때 상당히 어려운 부분이 많았음.

성공하니깐 뿌듯하다 ㅎㅎ


2.1에서는 설정한 값을 저장하는 코드를 대량 수정해버려서 2.0에서 설정한 기존 세팅 값이 날라가 버림.

쉽게 말하면 2.0에서 2.1로 버전을 올릴 시, 기존에 설정에서 설정한 값들이 전부 초기화 된다는 거임.

다음 버전부턴 초기화 안되고 잘 보존될 것임 ㅎㅎ


- 최적화

파일 크기가 약 40kb가량 줄어 듦. (2.0은 100kb)

코드 최적화로 프로그램이 작업을 처리하는 속도가 매우매우 미약하게 상승했을 거라고 생각함.


- 설정 기능 추가

'세션 관리자 실행시 자동으로 관리자 권한 요구 여부' : 이 옵션은 기존에 bat 파일을 우클릭해 관리자 권한으로 실행하는 애들을 위해 만들어짐.

더 이상 번거롭게 우클릭을 안하고 프로그램만 켜도 자동으로 관리자 권한으로 실행할 꺼냐 물어 봄,


'코드세션 전환 시 자동으로 소셜클럽 웹사이트 연결' : 이 옵션은 코드세션 전환 입력시 자동으로 소셜클럽 웹 사이트를 킬지 말지 설정할 수 있음.

항상 게임 끈 애들은 직접 소셜클럽 사이트에 들어가야 하는 불편함이 있었는데, 이 옵션을 사용하면 메인화면에서 3번을 입력시 바로 소셜클럽 웹사이트가 열려 불편함을 덜어줄 수 있음.


'Session Manager Info' : 미완성임.

프로그램 오류 나는 애들은 이 옵션을 사용해 나한테 로그를 보여주면 앵간한 건 다 고쳐줄 수 있을 것임.
